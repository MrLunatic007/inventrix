## ⚙️ Setup & Usage

    
### 1. Create a virtual environment
```bash
python -m venv venv
source venv/bin/activate
```

### 2. Install dependencies
```bash
pip install django
```

### 3. Run migrations
```bash
python manage.py migrate
```

### 4. Start the development server
```bash
python manage.py runserver
```

Now open your browser and visit:
👉 http://127.0.0.1:8000/
